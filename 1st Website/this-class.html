This Class

Welcome to my Intro to Computer Science webpage! Intro to Computer Science is all about understanding the basics of computers and getting a little more in depth about how they work. Since computers are a part of our everyday life, it is important for us to learn how they can be used and how we can use them creatively and to our advantage. So far, we have learned the very basics of HTML and CSS coding, which can be explained in another section above. Also in the tabs above, you can learn more about me and Computer Hardware. My classmates and I are very excited to be able to take this course and leave it with a better knowledge of Computer Science. We have been using the sites Codio and Git (so far) in order to be able to make web pages like this one. At first, when I saw Computer Science as one of our elective options, I was hesitant to choose it, because Computer Science seems really confusing and frustrating just by the description of it. But after giving it a try, I have realized that I actually really enjoy Computer Science and making stuff on my own.
